# 🎮 GameVerse Pro

GameVerse Pro is a modern gaming discovery web app built using **HTML, CSS, and JavaScript** with the RAWG API.  
It allows users to search games, explore trending titles, and filter by genres with a smooth and responsive UI.

---

## 🚀 Features

- 🔍 Search games using RAWG API  
- 🎯 Filter by genres (Action, Racing, Sports, RPG)  
- 🔥 Trending games section  
- 💡 Auto-suggestions while typing  
- 🌙 Light/Dark mode toggle  
- 📱 Fully responsive design (Mobile + Tablet + Desktop)  
- ❤️ Favorite button UI  

---

## 🛠 Tech Stack

- HTML5  
- CSS3 (Flexbox, Grid, Animations)  
- JavaScript (ES6+)  
- REST API (RAWG API)
